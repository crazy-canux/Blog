# ReStructuredText

reST是python的文档格式。格式类似markdown。

主标题：

    =====

副标题：

    -----

    # 表示部分
    * 表示章节
    =, 小章节
    -, 子章节
    ^, 次级子章节
    ", 段落

# Sphinx

Sphinx就是python处理reST格式的工具。

    pip install -U Sphinx
