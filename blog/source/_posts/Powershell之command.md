# cmdlet

列出所有指令和概念：

    get-help *

查看命令和概念的帮助：

    get-help <<CmdletName> <TopicName>>
    <CmdletName> -?

查看命令和概念的详细帮助：

    help <<CmdletName> <TopicName>> //
