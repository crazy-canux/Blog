# flask

<https://github.com/pallets/flask>

<http://flask.pocoo.org/>

<http://docs.jinkan.org/docs/flask/>

flask依赖两个库：

1. Jinja2

    渲染模板。

    <https://github.com/pallets/jinja>

2. Werkzeug

    WSGI工具集。

    <https://github.com/pallets/werkzeug>

# 安装

在virtualenv中安装：

    $mkdir mysite
    $cd mysite
    $virtualenv venv
    $. venv/bin/activate
    $pip install Flask

全局安装：

    sudo pip install Flask
